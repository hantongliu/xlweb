# Xue Lian Web

## Development

### Prerequisite
- [flask][flask]

If you have installed `python3` and `python3-pip`, you can run the following script to install `flask`.
```sh
pip3 install flask flask_cors
```

### Run
```sh
python3 app.py
```
Now the app should be running on port 5000.
